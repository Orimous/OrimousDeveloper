Планеты Солнечной Системы

Версия: 1.1

Узнай, что-то новое для себя о планетах

Приложение "Планеты Солнечной Системы" представляет приложение с приятным интерфейсом, в которой пользователь может изучить все планеты Солнечной Системы.

Пользователь может выбрать интересующую его планету и получить информацию о планете. Приложение "Планеты Солнечной Системы" - отличное приложение для всех любителей астрономии и желающих расширить свои знания об этой удивительной части космоса.
